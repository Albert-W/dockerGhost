# dockerGhost
It a web services which is build up by docker-compose.

效果如下
![image](ghost.png)

后台管理页:
![image](ghost2.png)

### 使用方式

```
docker-compose build
docker-compose up  // 可查看输出信息
docker-compose up -d // 后台执行
```

网站首页：
http://localhost:80/
或
http://localhost:2368/

后台管理页：
http://localhost:80/ghost
